Introduction au développement web

En tant que novice dans le domaine du développement web, je vais créer cette page pour me familiariser avec les concepts fondamentaux. Mon objectif est d'apprendre à structurer le contenu en utilisant HTML, ce qui inclut l'ajout de divers éléments tels que des titres, des paragraphes, des listes et des images. Cette expérience pratique me permettra de comprendre comment ces éléments interagissent pour former une page web cohérente et fonctionnelle. En progressant, je vais également explorer les bases de CSS pour styliser la page et JavaScript pour ajouter des fonctionnalités dynamiques.
